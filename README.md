## Wordly - Single Page Dictionary Application
Wordly is a modern, interactive dictionary interface designed for language learners. This Single Page Application (SPA) allows users to search for words and retrieve real-time data including definitions, pronunciations, and synonyms without a page reload.

## 🚀 Features
Live Search: Instantly fetch data from the Free Dictionary API.

Dynamic UI: Displays up to 3 definitions, part of speech, and source links dynamically.

Audio Playback: Listen to the correct pronunciation of searched words.

Theme Toggle: Switch between Light and Dark modes for better readability.

Responsive Design: Fully optimized for mobile, tablet, and desktop views using CSS Media Queries.

## 🛠️ Technologies Used
HTML5: Semantic structure for accessibility.

CSS3: Custom styling with Flexbox and Media Queries.

JavaScript (ES6+): Fetch API for data retrieval and DOM manipulation for real-time updates.

Public API: Free Dictionary API

## 📂 Project Structure
SearchIt.html - The main entry point and structure of the app.

style.css - Custom styles, including Dark Mode and Responsive Design logic.

script.js - Application logic, API handling, and event listeners.

## 🔧 How to Run
Clone the repository: git clone https://github.com/Ann-Annita23/Wordly_SPA

Open SearchIt.html in any modern web browser.

Enter a word and explore!
